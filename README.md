# DIG
## DI Generator
Generating Defective Interfering particles with 4 different methods


Requirements:
Python 3.0+ 
BioPython

Methods:
    ViReMa 
        Lengths with break point and reintination point 

    INDEL
        Lengths with 

    Copyback
        Break point

    MultiSeg
